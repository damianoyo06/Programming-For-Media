# Web Development Weekly Tasks

This repository contains weekly tasks completed during my Master’s program, focusing on web development using:

HTML
CSS
JavaScript

# Repository Structure
Each week has its own folder (Week 1, Week 2, etc.)
Each folder contains the tasks or mini-projects completed that week
Files are organised by task for easy navigation

# Purpose
Practice and reinforce web development concepts
Gain hands-on experience with front-end development
Keep a structured record of weekly progress

# How to Use
1. Clone or download the repository
2. Navigate to the folder for the week you want to view
3. Open the HTML files in a browser to see the output
4. Open the CSS and JavaScript files to review the code

# Notes
Tasks include exercises and mini-projects designed to improve your front-end skills
Modern browser support is recommended for best results
